## How It Works

```
SDK → OTLP/HTTP → Tracely API → Dashboard
```

1. **Install** the SDK: `pip install tracely`
2. **Initialize** with one line: `tracely.init()`
3. **Monitor** in the Pulse View dashboard — every request, query, and external call captured automatically.
